genesis_api_wrapper.catalogue module#

class genesis_api_wrapper.catalogue.CatalogueAPIWrapper(username: str, password: str, language: genesis_api_wrapper.enums.Language = Language.GERMAN)#

Bases: object

Create a new Wrapper containing functions for listing different object types

Parameters

  • username (str) – The username which will be used for authenticating at the database. Due to constraints of the database the username needs to be exactly 10 characters long and may not contain any whitespaces

  • password (str) – The password which will be used for authenticating at the database. Due to constraints of the database the password needs to be at least 10 characters long, may not exceed 20 characters and may not contain any whitespaces

  • language (enums.Language) – The language in which the responses are returned by the database. Language.GERMAN has the most compatibility with the database since most of the tables are on German. Therefore, this parameter defaults to Language.GERMAN

Raises

ValueError – The username or the password did not match the constraints stated in their description.

async cubes(object_name: str, storage_location: genesis_api_wrapper.enums.ObjectStorage = ObjectStorage.ALL, result_count: int = 100) → dict#

PREMIUM ACCESS REQUIRED

List the datacubes matching the object_name

Parameters

  • object_name (str) – The identifier code of the data cubes. The usage of an asterisk (*) is permitted as wildcard

  • storage_location (enums.ObjectStorage) – The storage location of the object, defaults to ObjectStorage.ALL

  • result_count (int) – The maximal amount of results which are returned by the database, defaults to 100

Returns

The response from the database parsed into a dict. If the Content-Type header indicated a non-JSON response the response is stored in a temporary file and the file path will be returned

Return type

dict, os.PathLike

Raises

exceptions.GENESISPermissionError – The supplied account does not have the permissions to access data cubes.
